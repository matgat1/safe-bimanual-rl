# D-ATACOM

This module is an adaptation of the D-ATACOM algorithm for MushroomRL 2.0.

The relevant code is selectively copied and adapted from the official repository:
[https://github.com/cube1324/d-atacom/tree/main](https://github.com/cube1324/d-atacom/tree/main)

## Scope

Only the components necessary for integrating D-ATACOM into this project are copied from the original repository. The code has been adapted to be compatible with **MushroomRL 2.0**, which differs from the version used in the original paper's codebase.
